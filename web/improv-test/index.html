pixel64 — Improv BLE test

A controlled Improv-over-BLE provisioning client. Use it to verify the device and to A/B the macOS write bug: write-with-response works; write-without-response is silently dropped by Chrome on macOS (improv-wifi/sdk-ble-js#213).

⚠️ This browser has no Web Bluetooth. Use Chrome or Edge on desktop or Android, over http://localhost or HTTPS. (Safari and iOS are unsupported.)
